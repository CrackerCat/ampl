### setup
/>mpl requires CMake and a C++ compiler to build, rlwrap is highly recommended for running the REPL.

```
$ cd ampl
$ mkdir build
$ cd build
$ cmake ..
$ make
$ rlwrap ./ampl
/>mpl v2

Press Return on empty line to evaluate.

May the source be with you!

```
